# Tools Directory

This directory contains various utility scripts and tools organized by category:

## Directory Structure

- **monitoring/**: Scripts for system monitoring and log management
  - `FINAL_WORKING_MONITOR.sh` - Main monitoring script
  - `monitor_autonomous_system_fixed.sh` - Fixed autonomous system monitor
  - `simple_monitor_test.sh` - Simple monitoring test
  - `clean_duplicate_logs.sh` - Log cleanup utility

- **database/**: Database-related scripts and SQL files
  - `diagnose_autonomous_db.sh` - Database diagnostic tool
  - `investigate_database.sh` - Database investigation script
  - `setup_analytics_tables.sql` - Analytics table setup
  - `fix_chat_aggregator_schema.sql` - Chat aggregator schema fixes

- **testing/**: Testing scripts and utilities
  - `test-chat-messages.js` - Chat message testing
  - `test-direct-chat.js` - Direct chat testing
  - `final-test-success.js` - Final test validation
  - `inject-test-message.js` - Message injection tool
  - `check-system-status.js` - System status checker
  - `test_log_parsing.sh` - Log parsing tests
  - `test_obs_realtime.sh` - OBS realtime tests
  - `test_rtmp_audio.sh` - RTMP audio tests
  - `validate_fixes.sh` - Fix validation
  - `debug_tool_parsing.sh` - Tool parsing debug

## Usage

Each subdirectory contains scripts related to its specific function. Make sure to review individual script documentation before running.
